# 🏛️ Personal Health Council

A personal health tracking web app built as a companion tool alongside real doctor consultations. Track vitals, pain, gout flares, exercise, diet, medications, and goals — all stored locally on your device.

## 🎯 Focus Areas

- **Gout Management** — Flare logging, trigger tracking, purine-aware diet guidance
- **Cardiovascular Health** — BP tracking, cholesterol/lipid monitoring
- **Sustainable Weight Loss** — Habit-based progress, not crash dieting
- **Pain Tracking** — 0-10 scale with traffic light system, body part logging
- **Medication Optimization** — Adherence tracking, dose history for doctor conversations
- **Healthy Aging** — Long-term vitality focus

## 🔒 Privacy

- **All data stays on your device** — stored in browser IndexedDB
- **No accounts, no cloud, no analytics** — zero tracking
- **Export as JSON/Markdown** — you own your data
- **Doctor Export** — generate structured visit summaries

## ⚕️ Disclaimer

This is a personal tracking companion, not a medical device or diagnostic tool. Always consult your healthcare providers before making changes to medications, diet, or exercise routines.

## 🛠️ Tech Stack

- Vanilla JavaScript + Web Components
- Dexie.js (IndexedDB)
- Chart.js (future)
- GitHub Pages — zero backend
